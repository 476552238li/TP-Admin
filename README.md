# TP-Admin
TP-Admin 是一个开源的，以ThinkPHP为底层架构的一个通用后台。

# 安装
下载最新版本的TP-Admin源码。
创建数据库，导入Docs/tp-admin.sql。
重命名Conf/database.example.php => Conf/database.php，并修改其中具体参数。
